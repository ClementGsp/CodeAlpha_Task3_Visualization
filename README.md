# CodeAlpha_Task3_Visualization
**Project Objective**
To transform raw agricultural data into an interactive visual format that enhances understanding of global farming strategies and supports data-driven decision-making.
**Tools Used:** Tableau
The visualizations show; a bar chart, a scatter plot and a dashbord with interactive crop filter. Thw bar chart reveals that the _Expansion of Harvested Area_ and _AI-based Advisory_ are the top global strategies, while the scatter plot shows government impact clearly dessigned to show correlation between government support and technology adoption indices
while the Interactive Crop Filter allows users to drill down into specific crop data, enhancing the dashboard's utility for strategic decision-making.
